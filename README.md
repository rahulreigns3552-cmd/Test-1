# Mista Eats - React (Vite) Clone

A responsive, multi-page React site inspired by Mista Eats. Built with Vite, React Router v6, functional components and CSS Modules.

Run locally:

```bash
npm install
npm run dev
```

Pages:
- Home (/)
- Plans (/plans)
- Diet Plans (/dietplans)
- About (/about)
- Contact (/contact)
- Shipping Policy (/shipping-policy)
- Terms & Conditions (/terms)

Project structure: `src/pages`, `src/components`, `src/styles`, `src/assets`.
